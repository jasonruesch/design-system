Badges are compact labels for status, category, or count. Variant colors are
driven entirely by design tokens, so they adapt to the active theme and brand
automatically.

## When to use

- **Use a Badge** to convey status (Success, Warning, Danger) or tag content
  with a category or count at a glance.
- Keep the label short — one to three words. For longer contextual messages,
  reach for an Alert or inline text instead.
- Avoid using badges as interactive controls; they are display-only.

## Accessibility

- Renders a native `<span>`, so it carries no implicit ARIA role. If the badge
  conveys meaningful status, pair it with a visually hidden label or add
  `aria-label` on the surrounding element to describe the context.
- Badge text is always visible — do not rely on color alone to communicate
  meaning. Pair the color variant with a word or icon.
- Because `<span>` is not focusable, badges require no keyboard interaction
  handling.
